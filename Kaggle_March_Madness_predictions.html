Visualizing your Kaggle March Madness predictions

Select your Kaggle March Madness submission file from your local file storage to visualize it. For the Men's tournament, you can visualize any file that's valid for a year from 1985 through 2019. For the women - any year from 1998 through 2019. Alternatively, you can view an example by selecting from the Examples menu.

The layout generally seems to be better in Firefox than it is in Chrome, particularly for the women's tournaments. I'm not sure why.

A little more info

If you know what a "Kaggle March Madness submission file" is then, hopefully, this page is a little fun. If not, you can read about it on Kaggle's page for the men's competition or their page for the women's competition.

After submission, the page should display a bracket with games that have occurred shaded according to your prediction. If you predicted the winner with probability less than 1/2, the game should be shaded red. If you predicted the winner with probability greater than 1/2, the game should be shaded green. You can hover over the game to get the precise prediction.
